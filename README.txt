== Description ==
El plugin **[LFECFDI para WooCommerce](https://lasfacturaselectronicas.com/factura-electronica-cfdi-woocommerce/)** es una herramienta que se integra a LFE Web y te permite llevar a cabo el proceso facturacion electronica de las ventas realizadas a tus clientes con WooCommerce en tu sitio web, almacenando tus CFDI en LFE Web con la posibilidad de administrarlos tambien en el panel del plugin en Wordpress.

Ventajas del plugin

* Consulta de facturas emitidas
* Cancelacion de facturas
* Acuse de cancelacion
* Descarga del PDF y XML
* Autofacturacion para el cliente
* Autocarga de datos de facturacion

== Installation ==

=== Desde WordPress ===
1. Visita 'Plugins > Anadir nuevo'
1. Busca 'LFECFDI para WooCommerce'
1. Activa el plugin desde la pagina de activacion.
1. Consulta la seccion de 'Despues de activar'.

=== Manualmente ===
1. Sube el folder `lasfacturaselectronicas_woocommerce` al directorio `/wp-content/plugins/`
1. Activa el plugin 'LFECFDI para WooCommerce' desde la pagina de activacion.
1. Consulta la seccion de 'Despues de activar'.

=== Despues de activar ===

**[Consulta el manual](http://comprobante.realvirtual.com.mx/comprobante_digital/plugin_lfe/Manual_LFECFDI-para-WooCommerce.pdf)**
